Changelog
#########

The first production LimeRFE was revision v1.0 and so this changelog starts with changes from that point.

.. note::
   Details of internal-only versions which were never released are not included here.

v1.31
****

Here is a list of changes between 1v0 and 1v31:

1. Additional SAW filters with matching networks have been added:

   Cell Band 01: U160

   Cell Band 02: U164, L121

   Cell Band 03: U165, L122, L123
   
   Cell Band 07: U166, L124, L125

2. GPIO pins have been modified so that GPIO0 and GPIO1 now support Arduino interrupt routines (IR).

   LNA_EN and PA_ENn have been moved at the MCU from pins 1 and 32 to pins 14 and 13, respectively.

   GPIO1_MCU and GPIO0_MCU are now connected to MCU interrupt pins INT1 and INT0 (pins 1 and 32), respectively.

3. To support TDD operation without an interrupt routine (IR), U167 has been added.

   An inverter was also added to make BAND38_TX_ENn active low.

4. R138 has been added so that the ICSP is directly connected to +5V_MCU node.

5. Since the PE4245 was not available on the market, it has been replaced with the HMC544AE.
   
   Affected components: U61, U63

   Control signals for the new switches are generated by U137. These control signals are now 3 V instead of the previous 2.5 V.

6. The HAM3500 PA designated as MAAM-009560 has been replaced with the MMZ38333, and the surrounding circuitry has been modified accordingly.

7. The section between U57 (cellular predriver) and U86 (RF switch) has been changed from:

   MADL-011021-14150T (U160, limiter) 

   PAT1220-C-3DB-T5 (U161, attenuator)

   to:

   PAT1220-C-3DB-T5 (U149, not fitted, attenuator)
   
   MADL-011008-14120T (U150, limiter)

   PAT1220-C-5DB-T5 (U161, not fitted, attenuator)

   Additionally, C150, C151, and L120 have been added.

8. GPIO0, GPIO1, GPIO4 and GPIO5 have been changed, which caused additional GPIO pins remapping on several connectors and components.

   GPIO remapping on Q3–Q6, U27, U28, SW2, J13, and U23–U26:
   from GPIO0, GPIO1, GPIO2, GPIO3 
   to GPIO2, GPIO3, GPIO4, GPIO5.

   This also remapped RELAY_0, RELAY_1, RELAY_2, and RELAY_3
   from GPIO0, GPIO1, GPIO2, GPIO3
   to GPIO2, GPIO3, GPIO4, GPIO5.

   J22 connector remapping:
   GPIO5, GPIO3, GPIO3_CONN changed to GPIO1, GPIO5, GPIO5_CONN.

   J12 connector remapping:
   GPIO0, GPIO1, GPIO2, GPIO3_CONN, GPIO4, GPIO5
   changed to GPIO0, GPIO1, GPIO2, GPIO3, GPIO4, GPIO5_CONN.

9. RF switch U86 connection to TX Cellular band 7 changed from pin RF4 to RF6.
