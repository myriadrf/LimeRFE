LEDs
####

The LimeRFE board includes ten indicator LEDs. Their positions on the board are shown in Figure 9.

.. figure:: /images/LimeRFE_v1.0_components_LEDs.png
  :width: 600

  Figure 9. LimeRFE v1.0 indication LEDs 

LEDs funcions are described in table4.

.. list-table:: Table 4. Default LEDs configuration
   :header-rows: 1
   :stub-columns: 1

   * - Board Reference
     - Description
   * - D1, D2
     - USB to serial communication indicator
   * - D3
     - +3V power indicator
   * - D7
     - +5V_MCU power indicator
   * - D9
     - +7.5V power indicator
   * - D10
     - GPIO0 indicator
   * - D11
     - GPIO2 indicator
   * - D13
     - GPIO1 indicator
   * - D14
     - GPIO3 indicator
   * - D36
     - +2.5V power indicator
